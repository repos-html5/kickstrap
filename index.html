Check out the source of this page, index.html in your Kickstrap root to see how some of this magic is done.

If instead you'd like to jump right in, use sample_index.html, a (nearly) blank html document with all the default Kickstrap magic linked and ready to go.

To make css changes, your best bet is to download a less compiler and mess around in the /extras directory.

See the Documentation to learn more.

In an effort to make Kickstrap run as smoothly and quickly as a regular version of Bootstrap, we've made some of the goodies which add to the page load optional. For example, to use js plugins, you'll need to add them in your markup per the Documentation

This is also a good time to start thinking of the production of your website. Do you want to minify and compress your css and js? If so, consider using the included ant build script in the build folder.

The version you're looking at now is a release candidate. That means two important things:

(1) This should not be used for production.

(2) I want to hear Feedback from developers to make this as useful and bug free as possible.

Thanks for checking this out.

Enjoy!
